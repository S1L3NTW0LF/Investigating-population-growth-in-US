def transform(col):
    col = col.str.replace(',', '').str.replace('$', '')

def to_percent(col):
    col = col.str.replace('%', '')
    col = pd.to_numeric(col)
    col = col / 100
    return col

Conclusions¶

From examining how the population, proportion of foreign born population, and median household income effect population growth it can be determined that there is not a strong correlation between any of them. The % of foreign born population had the clearest correlation and it would be safe to assume that the other variables had positive if not weak correlations. To make the results of this analysis more accurate it would be better if the data collected was all from the exact same dates. Median household income was collected from 2011 to 2015 and the other variables were collected from 2016. If there was a major event that happened during 2016 it may change the household income data noticeably.

Limitation¶

The limitations of this data are that the data for population did not take into account how large the state was which would eliminate population density as a factor that could be affecting the results being produced, and for household income it would be more accurate to use data that is the average income of a person in that state because there may be differences in the type of households between states.
